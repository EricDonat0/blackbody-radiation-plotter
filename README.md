# Simulação da Lei de Planck em Python

Este projeto implementa a Lei de Radiação de Planck para calcular e visualizar a distribuição espectral de um corpo negro em diferentes temperaturas. O objetivo é demonstrar como a radiância varia com o comprimento de onda e como curvas de diferentes temperaturas se comportam.

---

## Descrição

O programa:

- Calcula a radiância espectral usando a **Lei de Planck**.
- Plota as curvas de emissão para várias temperaturas.
- Marca o comprimento de onda de máxima emissão utilizando a **Lei de Wien**.
- Calcula a intensidade total emitida por um corpo negro com base na **Lei de Stefan–Boltzmann**.

Ele é útil para fins educacionais e demonstrações sobre radiação térmica em Física Moderna.
